[SP]----------------------------------------------------------

La librería sirve para hacer parpadear un led durante el tiempo que queramos.

·Declarar objeto:

	Blink nombre (pin). //"pin" es el pin de Arduino al que está conectado.
	
	Ejemplo ---> Blink led1 (2);

·Llamar a la función: 

	name.parpadear(time); //"time" en milisegundos.

	Ejemplo ---> led1.parpadear(500);

[EN]----------------------------------------------------------

The library is used to flash a LED for as long as we want.

·Declare object:
	
	Blink name (pin); //"pin" is the Arduino pin to which it is connected.

	Example ---> Blink led1 (2);

·Call the function:

	name.parpadear(time); //"time" in ms.

	Example ---> led1.parpadear(500);
